# 設計規劃書：修行加速攻略（分級指南系統）

> 目的：引導學員快速理解計分機制，掌握「如何高效拿分、進化神獸、構築牌組」，並根據學員身處「初階」或「進階」期數，自動提供最適配的修煉指引，提升參與感與定課黏著度。

---

## ── 視覺設計風格 ──
我們建議延續本系統極致高階的 **暗黑玻璃微光風格 (Glassmorphism)**，並在不同難度間進行色彩分流：
*   **🟢 初階日常攻略**：採用 **「翡翠綠 ➔ 冰晶藍」** 的微光漸層，象徵初學、萌芽與習慣建立。
*   **🔥 進階修煉心法**：採用 **「烈焰橘 ➔ 玄金黃」** 的烈焰漸層，象徵突破、高階挑戰與團隊榮譽。

---

## 一、 前端 UI 入口與版面設計

### 1. 入口位置
*   **學生控制面板**：在首頁「每日定課」與「神獸狀態」卡片上方，新增一個亮眼的浮動金屬質感徽章或橫幅按鈕：`⚡ 修行加速攻略`（伴隨輕微的流光微動畫）。
*   **點擊行為**：彈出一個全螢幕的玻璃磨砂 Modal（或做成一個獨立的頁面分頁 Tab），方便學員在手機上滑動閱讀。

### 2. 佈局結構
*   **頂部切換鈕**：放置一組精緻的 Segmented Tab（初階定課指南 / 進階修煉心法）。
    *   **自動預判**：系統讀取學員當前班次的名稱（例如 `userBatch.name`）。若含有「進階」、「高階」或「班長班」等字眼，預設自動選取「進階修煉心法」；其餘預設開啟「初階日常指南」，但學員仍可手動切換閱讀。
*   **主體內容區**：採用卡片式排版（Grid Layout），每張卡片代表一個「加速要訣」，包含 Icon、標題、核心分數加成、以及「立即前往」的引導按鈕。

## 二、 攻略內容規劃（初階 vs 進階）

### 🟢 初階日常指南（核心：建立習慣，穩拿基礎分）

針對初階學員，攻略應專注於**「不漏分」**與**「熟悉系統操作」**：

| 加速要訣 | 攻略心法 | 前端快捷導向按鈕 |
| :--- | :--- | :--- |
| **1. 每日定課簽到** | 每天自動審核的基礎任務（如每日靜心、定課閱讀）是得分地基，**連續完成 7 天有額外加成**，切記不可中斷。 | `[🎯 去做任務]` |
| **2. 混沌之卵孵化** | 經驗值每達 **700分** 神獸即可升級。升到 **5級** 時，將自動解鎖進化。越早解鎖，越能享受神獸帶來的被動加成。 | `[🥚 查看我的神獸]` |
| **3. 見證牆初試啼聲** | 將有深刻體會的定課成果，勾選「分享至見證牆」，不僅能記錄自己的蛻變，**獲得隊友點讚還能額外增加人氣分**。 | `[✍️ 去寫見證]` |
| **4. 掃碼加入小隊** | 確認自己已加入正確的小隊。**小隊長每日手動調分與出席登記**是不可忽視的隱藏分數來源。 | `[👥 我的小隊]` |

---

### 🔥 進階修煉心法（核心：突破極限，追求團隊榮譽與策略卡牌）

針對進階學員，攻略側重於**「高分任務、神獸方向選擇、卡牌組合配對」**：

| 加速要訣 | 攻略心法 | 前端快捷導向按鈕 |
| :--- | :--- | :--- |
| **1. 挑戰高階審核任務** | 進階期數會開放需要小隊長或大隊長審核的「實踐影音」、「深度心得」等高難度任務，**單次分數是一般任務的 3~5 倍**，是拉開差距的關鍵。 | `[🔥 挑戰高分任務]` |
| **2. 小隊全員共創加成** | 當小隊每日打卡率達到 **100%** 時，全隊成員可獲得額外的「同心協力加分」，這需要小隊長與組員在群組中緊密提醒。 | `[🏆 排行榜動態]` |
| **3. 四大神獸流派選擇** | 達到 5 級時，會出現 4 個神秘的「剪影進化方向」（水、火、風、地）。學員須先去完成對應的**進化考驗任務**並獲准，才能進行「破殼儀式」，開啟不同的加分流派。 | `[🐉 進行破殼儀式]` |
| **4. 神兵卡牌牌組構築** | 利用修煉得到的經驗解鎖卡牌，並在「牌組」頁面配置您的主動/被動卡牌。**配置 SSR 或同元素卡牌（如全火系牌組）會觸發「屬性共鳴」，讓您每次打卡的分數額外加成 10%~20%**！ | `[🃏 構築我的牌組]` |

## 三、 技術實作方案

### 1. 靜態資料結構化
為確保攻略頁面「秒開」、不增加資料庫查詢負擔，所有攻略資料應儲存在前端的靜態設定檔 [lib/guideConfig.ts](file:///Users/leo/Desktop/定課系統/NLP_GAME/lib/guideConfig.ts) 中：

```typescript
export interface GuideItem {
  id: string;
  title: string;
  scoreLabel: string; // 例如: "+50 XP" 或 "高達 3倍"
  description: string;
  actionText: string;
  actionTab: string; // 點擊按鈕後要切換的前端 Tab 名稱
}

export const BEGINNER_GUIDES: GuideItem[] = [
  {
    id: 'b1',
    title: '每日定課簽到',
    scoreLabel: '每日穩拿 50+ 經驗',
    description: '每天自動審核的基礎任務是得分地基，連續完成有額外成就獎勵...',
    actionText: '去做任務',
    actionTab: 'daily'
  },
  // ...其餘
];

export const ADVANCED_GUIDES: GuideItem[] = [
  {
    id: 'a1',
    title: '四大神獸流派選擇',
    scoreLabel: '解鎖專屬加分流派',
    description: '達到 5 級時，挑戰水火風地四種考驗任務，通過即可完成破殼進化...',
    actionText: '進行破殼',
    actionTab: 'quests' // 導航至進化面板
  },
  // ...其餘
];
```

### 2. 前端介面渲染與導航
*   在 `app/page.tsx` 或新 Tab 元件中，使用一個 `activeGuideLevel` 狀態 (`'beginner' | 'advanced'`) 渲染對應的陣列。
*   點擊按鈕時，調用 `setActiveTab(item.actionTab)`，系統會自動平滑切換到相應的功能區（例如學員點擊「構築牌組」，頁面會立即流暢地跳轉至卡牌 Tab），達到極致的互動體驗。
