# 目錄說明（中文對照表）

> 檔名維持英文（供 Router / Skills / 交叉引用穩定指路），本表提供中文對照。
> 每支檔案開頭也都有中文標題，可直接開檔閱讀。
>
> **對帳說明（R8 二選一）**：`.claude/skills/` 表由 preflight 第 13 項與實際資料夾機器對帳；
> 其餘各表為手動維護，**可能落後實況，以資料夾實際內容為準**。
>
> **熱/冷分層**：「熱」＝AI 日常必讀、維護與潤飾優先；「冷」＝情境參考，出事或碰到該領域才查。
> **冷≠沒用**——它們是正確的參考書，只是維護優先度不同。

## scripts/ — 機器護欄

| 檔案 | 中文名稱 | 用途 |
|---|---|---|
| `preflight-check.sh` | 紅線掃描腳本 | commit 前掃 using(true)、機密外洩、既有 migration 被改等 |
| `install-hooks.sh` | Hook 安裝腳本 | 在新專案把 preflight 掛成 pre-commit hook |
| `release.sh` | 發版一致性閘門 | `bash scripts/release.sh vX.Y.Z`：只在 changelog／檔頭／版號一致且工作區乾淨時才打 tag，防幽靈版本 |
| `review.sh` | 高風險獨立審查 | 取 `git diff origin/main..HEAD` 交乾淨 AI 實例以資安+QA 視角審，輸出 `logs/reviews/<sha>.md`（軟跑階段） |
| `preflight-allow.txt` | RLS 例外白名單 | 登記 migration 中已審核放行的政策（路徑:行號） |
