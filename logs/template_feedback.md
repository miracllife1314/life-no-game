# Template Feedback — 模板回饋暫存

> 用途：在**本專案開發過程中**,累積「模板哪裡好用 / 哪裡卡 / 缺什麼」的回饋。
> 每完成一個功能或收工時,執行下方 retro prompt 讓 AI 幫你收集。
> 週期性(每完成大功能或每週)把本檔帶回「萬用AI系統開發2版」母版對話,評估要不要回流升版。
> 詳細流程見 `docs/FEEDBACK_LOOP.md`。

---

## 收工 Retro Prompt（複製即用）

```text
回顧這次開發，用資深工程師視角誠實回答：
1. 有沒有哪條規範幫我擋掉了問題？（值得保留）
2. 有沒有哪條規範卡到我、但其實沒必要那麼嚴？（可能太嚴）
3. 有沒有哪個風險是規範沒擋到、我事後才發現的？（可能缺規則）
4. 我有沒有重複糾正你同一件事？（該固化成規則）
5. preflight 有沒有誤報或漏報？
把每項寫成一句「模板改進建議」，記到 logs/template_feedback.md。
```
