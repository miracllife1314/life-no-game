# NLP Game - 專案版本發布記錄 (v1.2.0)

本版本 (v1.2.0) 已正式提交至 Git 本地倉庫做版本存檔。以下為此版本的主要功能、版面樣式優化及修復內容摘要：

---

## 🚀 新增功能與後台管理

1. **課程報名連結按鈕與預設連結**：
   - 學生端的「課程」分頁現已針對所有課程卡片**強制顯示「前往課程報名連結」按鈕**。
   - 若管理員發布課程時未輸入特定網址，按鈕將會自動採用預設報名連結 (`https://example.com/register-nlp`)，確保視覺體驗一致。

2. **後台發布課程管理與刪除功能**：
   - 於管理員後台的「發布課程與日期」區塊下方，新增了「已發布課程列表」。
   - 管理員可在此檢視所有已發布課程的名稱、日期、對應期數及報名連結，並可使用「垃圾桶」按鈕進行一鍵刪除與重新發布。

---

## 🎨 版面視覺與對比度優化 (高對比度防護)

1. **移除課程日期徽章**：
   - 移除了卡片上重複顯示的黃色「課程日期：YYYY-MM-DD」徽章，使版面回歸純粹的課程資訊與報名功能。

2. **標題顏色換成琥珀色系**：
   - 課程名稱標題改用高質感琥珀金色。深色模式下為亮琥珀色 (`text-amber-500`)，淺色模式下為深琥珀金 (`light:text-amber-800`)，確保與背景有足夠的高對比度。

3. **雙主題文字對比度提升 (底色與字體更清晰)**：
   - 提升課程敘述、小隊徽章與通知列文字在淺色主題下的色彩對比度，解決原本灰色字體偏淡、難以閱讀的問題。

4. **修正深紫色按鈕字體顏色**：
   - 解決了淺色模式下全域規則導致的「紫底黑字」對比度缺陷。
   - 新增了全域 CSS 強制覆蓋規則，確保 `bg-purple-600`、`bg-blue-600` 與 `bg-red-600` 等彩色實色按鈕在任何模式下均呈現**純白文字 (`#ffffff`)**。

---

## 👥 小隊與角色邏輯優化

1. **更正小隊長職稱**：
   - 將標頭（Header）處的 `小隊長・導師` 標籤更正為符合大會設定的 **「小隊長」**。

2. **小隊命名格式優化與自動移轉**：
   - 首頁標頭（Header）的小隊名稱顯示順序調整為 **`預設隊名 (自訂隊名)`**（例如：`NLP初階50期第二隊 (帥氣對)`）。
   - 在標頭中**自動動態切除重複的期數前綴**，例如將 `NLP初階50期第二隊 (帥氣對)` 簡化為：`第二隊 (帥氣對)`，版面更為精簡。
   - 在 `lib/supabase.ts` 中實作了全自動的本地資料庫（localStorage）升級移轉，會自動修復舊版本被自訂名稱覆蓋的原小隊名稱，並安全地將自訂小隊名稱存入專屬欄位。
